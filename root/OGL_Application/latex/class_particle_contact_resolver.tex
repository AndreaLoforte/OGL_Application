Resolves a set of particle contacts for both penetration and velocity.

Contacts that cannot interact with each other should be passed to separate calls to resolve\+Contacts, as the resolution algorithm takes much longer for lots of contacts than it does for the same number of contacts in small sets.


\begin{DoxyParams}{Parameters}
{\em contact\+Array} & Pointer to an array of particle contact objects.\\
\hline
{\em num\+Contacts} & The number of contacts in the array to resolve.\\
\hline
{\em num\+Iterations} & The number of iterations through the resolution algorithm. This should be at least the number of contacts (otherwise some constraints will not be resolved -\/ although sometimes this is not noticable). If the iterations are not needed they will not be used, so adding more iterations may not make any difference. But in some cases you would need millions of iterations. Think about the number of iterations as a bound\+: if you specify a large number, sometimes the algorithm W\+I\+LL use it, and you may drop frames.\\
\hline
{\em duration} & The duration of the previous integration step. This is used to compensate for forces applied. \\
